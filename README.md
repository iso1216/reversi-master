# MINE-SWEEPER

Ver 1.0.1 (2024/4月時点)

## 概要

- プロダクト名：reversi-master
- 使用言語：JavaScript(React)、HTML/CSS
- 開発環境：VSCode
- CSS設計：MUIを使用(使いやすく、シンプルな見た目で作成がしやすい為)

## サイト内容

- トップページは二人で対戦(Start Game)かCPUと対戦(VS CPU)のどちらかを選択します。
- 選択したモードに合わせてゲームがプレイできます。
